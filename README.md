# Counter

A simple Godot-based screen counter with keyboard input via a Python helper script. Named values can be stored under categories, then linked to the on-screen counter.

---

## Requirements

- [Python 3](https://www.python.org/)
- [`pynput`](https://pypi.org/project/pynput/) library

Install `pynput` via pip:

```bash
pip install pynput
```

---

## Usage

1. **Launch the app** (`App.exe`).
2. When the counter window is focused:
   - Press the **arrow keys** or **hold the left click and drag** to move the window.
   - **Right-click** on the window to access the options menu.
  
3. From the options menu, you can:
   - Open the **Values Editor** window to create categories and named counters, then link one to the main counter.
   - Reset the current value
   - Quit the app 

4. The Python script listens for key presses and sends them to the Godot app via UDP.
   - Press **`+`** to increment the counter.
   - Press **`-`** to decrement the counter.
   - Press  **`F9`** to lock or unlock the counter window, making the mouse passthrough.

## Project Structure

```
/project
├── App.exe
└── files
    ├── inputs_handler.py   # Python script that handles '+' , '-' and 'F9' key input
    ├── datas.csv           # CSV file generated by the 'values editor' window
    └── conf.cfg            # Configuration file generated by the app (window pos, last value, last_visited_category and linked_entry)
```

---

## Notes

- The Python script is launched by the Godot app on startup.
- Python script (`inputs_handler.py`) can be edited to change keys.
- Make sure no firewall blocks UDP communication on the configured port (default: `9090`).
